# Postal Click

Postal Click es una aplicación web para obtener tu código postal fácil y rápido.

## Características principales
- Búsqueda dual: puedes obtener el código postal haciendo click en el mapa o ingresando una dirección.
- Multilingüe: soporte para el español, inglés y portugués.
- Diseño responsivo: optimizado para escritorio y dispositivos móviles.
